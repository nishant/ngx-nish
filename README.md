# NgxNish

A personal component + design system library for Angular.

## Components
* Banner
* Breadcrumbs
* Button
* Checkbox
* Footer
* Modal
* Navbar
* Page Section
* Page Section Divider
* Radio Button
* Ribbon
* Video Player

## Styling
* CSS Reset for browser uniformity
* Generic element stylesheet
* Light mode stylesheet
* Dark mode stylesheet
* Design Tokens
  - Color palette
  - Spacing
  - Typography
  - Responsive breakpoints
